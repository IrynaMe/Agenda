Crea un'applicazione web Spring che gestisca una lista di appuntamenti.
Ogni appuntamento dovrebbe avere un titolo, una descrizione e una data.
 L'applicazione dovrebbe permettere di
 visualizzare tutti gli appuntamenti,
 aggiungere nuovi appuntamenti,
 eliminare appuntamenti esistenti e
 modificare i dettagli degli appuntamenti esistenti.
